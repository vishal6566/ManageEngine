ManageEngine AD360 is an integrated identity and access management (IAM) solution for managing user
identities, governing access to resources, enforcing security, and ensuring compliance. From user
provisioning, self-service password management, and Active Directory change monitoring, to single sign-on
(SSO) for enterprise applications, AD360 helps you perform all your IAM tasks with a simple, easy-to-use
interface.
